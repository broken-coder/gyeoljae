# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, no harassment of any kind. Disagreements about code are fine; disrespect toward people is not.

To report a violation, contact the maintainer privately via GitHub ([@broken-coder](https://github.com/broken-coder)). Reports are handled confidentially.
